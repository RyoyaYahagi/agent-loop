## PR 本文テンプレート（日本語）

このテンプレートを使用して、PR本文を作成する。

```markdown
## 目的

（Issueの解決する問題や、追加する機能の1行要約）

## 変更内容

- 変更点1
- 変更点2
- 変更点3

## 実行したテスト

- [ ] `npm run typecheck` — 通過
- [ ] `npm run lint` — 通過
- [ ] `npm run format:check` — 通過
- [ ] `npm test` — 通過

## 残タスク

- （必要なら記載）

## 人間に確認してほしい点

- （アーキテクチャ、コードレビュー、ビジネスロジックなど）

Closes #<issue-number>
```
